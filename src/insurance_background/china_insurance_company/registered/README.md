# 官方备案的保险公司

## 中国现有备案的保险公司

* 前面已述：保险公司`牌照`是**稀缺资源**
  * 保险法规定至少要有<font color=red>2亿</font>元人民币实缴资本
  * 现实中没有<font color=red>20亿以上</font>真金白银是很难办成这事的
    * 即使这20亿元不是问题，如果出资股东背景鱼龙混杂，那拿牌照一样不会是容易事
