# 意外险

* 注意
  * 保险中的`意外险`都是`一年一签`的
    * 且每天都可能变更，甚至**停售**

## 停售 举例

比如：

之前买的招商信诺的主险的附加险：

《招商信诺附加残疾意外伤害保险》

```bash

```
